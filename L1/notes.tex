\section{What is a Stochastic Process}
A stochastic process is some random variable $X_t$ whose value changes over time interval $t$. In continuous time, $t \in \mathbb{R}$. In discrete time, $t\in [1, 2, \dots k]$ for some $k \in \mathbb{R}^+$ . The outcomes of $X_t$ at different $t$s can be $\textit{correlated}$ with each other. $X_t$ can take values in any space S (real line, d-dimensional euclidean space, non-euclidean manifold)
